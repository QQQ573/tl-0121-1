## 1. 产品概述

标准化考场 3D 视线预览系统——为考场改造方案提供可视化验证工具，帮助考务人员、家长直观评估最后一排考生能否看清黑板，替代仅有的平面座次表。
- 解决家长投诉"最后一排看不清黑板"的痛点，将视线遮挡从经验判断升级为精确可量化的 3D 分析
- 目标用户：考务管理人员、学校后勤、家长委员会

## 2. 核心功能

### 2.1 用户角色

| 角色 | 使用方式 | 核心权限 |
|------|----------|----------|
| 考务人员 | 直接访问 | 调整参数、切换预设、分析所有座位视线 |
| 家长 | 直接访问 | 查看 3D 预览、点击座位查看可视比例 |

### 2.2 功能模块

1. **3D 考场预览页**：3D 教室场景、课桌椅网格、黑板讲台、视线射线可视化、座位遮挡状态

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 3D 考场预览页 | 参数控制面板 | 输入行数、列数、行间距、列间距、讲台高度、黑板宽度；切换排布预设 |
| 3D 考场预览页 | 3D 场景画布 | 渲染教室、课桌椅、黑板、讲台；Orbit 旋转/缩放/平移 |
| 3D 考场预览页 | 座位交互 | 点击座位选中，从该座位眼睛位置向黑板发射简化视线射线 |
| 3D 考场预览页 | 遮挡分析 | 被前排头影遮挡的座位垫标红，信息面板显示可视黑板比例 % |
| 3D 考场预览页 | 排布预设 | 单列、双列、梅花（隔行错位）三种排布一键切换 |

## 3. 核心流程

用户打开页面 → 输入/调整考场参数（或选择预设） → 系统自动生成 3D 课桌椅网格 → 用户拖拽旋转/缩放查看整体布局 → 点击任意座位 → 系统从该座位眼睛高度向黑板边缘发射视线射线 → 检测是否被前排考生头影遮挡 → 座位垫颜色反馈（红=遮挡/绿=清晰）→ 显示该座位的可视黑板比例 → 用户调整参数优化布局

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：深青 (#0D1B2A) + 琥珀黄 (#F5A623) 作为强调色，营造专业考场氛围
- 辅助色：海蓝 (#1B3A4B)、浅灰白 (#E0E1DD)
- 按钮：圆角微 3D，琥珀黄 hover 高亮
- 字体：Noto Sans SC（中文），JetBrains Mono（数值参数）
- 布局：左侧参数面板 + 右侧 3D 画布，底部信息条

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 3D 考场预览页 | 参数控制面板 | 深色侧边栏，数值输入框+滑块，预设按钮组，参数实时联动 |
| 3D 考场预览页 | 3D 场景画布 | 占满右侧区域，浅色地面网格，柔和环境光+方向光 |
| 3D 考场预览页 | 遮挡分析信息 | 底部浮层，座位编号+可视比例+遮挡描述，淡入动画 |
| 3D 考场预览页 | 视线射线可视化 | 从选中座位眼点到黑板边缘的彩色射线，遮挡段红色高亮 |

### 4.3 响应式设计

- 桌面端：左侧面板 + 右侧 3D 画布并排布局
- 平板/移动端：面板折叠为顶部工具栏抽屉，3D 画布全屏
- 移动端双指缩放/旋转，支持触摸选座

### 4.4 3D 场景指引

- 环境：室内教室场景，无 HDRI，用暖白方向光模拟日光
- 灯光：AmbientLight (0.6) + DirectionalLight (0.8) 从侧上方投射，营造教室明亮感
- 相机：初始透视相机位于教室后上方 45° 俯瞰，OrbitControls 可自由旋转
- 焦点元素：选中的座位有发光高亮环，视线射线为醒目彩色线条
- 交互：点击座位选中 → 射线生成 → 遮挡分析动画 0.5s 过渡
- 性能预算：60fps 流畅交互，课桌椅使用 InstancedMesh 批量渲染
